# Filter

Shows why naive post filter is not effective for ANN index such as faiss IVF.
Use tantivy to index IVF cluster id for vectors in attributes index.

## Usage

```bash
# SentenceTransformer requires ipython when using marimo
pip install marimo ipython
marimo edit
```
